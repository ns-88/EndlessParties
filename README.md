## 📝 Описание проекта
EndlessParties - простой сервис для организации мероприятий любого уровня: от встреч в верхах до дружеских посиделок у костра

## 🛠 Возможности API
API предоставляет следующие возможности для управления мероприятиями (Events):

- **GET /events**: получение списка всех событий
- **GET /events/{id}**: получение события по идентификатору
- **POST /events**: создание события
- **PUT /events/{id}**: обновление события
- **DELETE /events/{id}**: удаление события

## 🚀 Сборка и запуск

### Предварительные требования
- .NET 10 SDK
- IDE (Visual Studio 2022 / JetBrains Rider / VS Code)

### Запуск
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/ns-88/EndlessParties
   ```
2. Перейдите в директорию проекта:
   ```bash
   cd EndlessParties
   ```
3. Восстановите зависимости и соберите проект:
   ```bash
   dotnet build
   ```
4. Запустите приложение:
   ```bash
   dotnet run --project src\Presentation\EndlessParties.Presentation
   ```

После запуска Swagger-документация будет доступна по адресу: `http://localhost:5000/index.html`
